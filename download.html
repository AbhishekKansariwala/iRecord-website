System Requirements

Check Before You Download

i-Record runs on standard Windows machines. Verify your PC meets these minimum requirements for the best experience.

Minimum System Requirements

Processor1.8 GHz x86-64 — Intel Pentium, Core 2 Duo, Dual Core, i3, i5, i7 or equivalent / above
RAMMinimum 8 GB or more
Hard Disk300 MB free space for iRecord app · 1.5 GB free for MS SQL version setup
Display1366 × 768 resolution or higher
Operating System64-bit editions of Windows 10 / Windows 11 / Server 2016 / Server 2022 & above
Microsoft SQLMicrosoft SQL Server 2014 / 2019
.NET Framework3.5.1 (SP1), 4.5.2 & above

Compatible OS Versions

  • Windows 10 (64-bit) — Recommended
  • Windows 11 (64-bit) — Fully Supported
  • Windows Server 2016 / 2022
  • Windows Server 2012 (not for MS SQL 2019)

Before You Install

  • Ensure Windows is up to date
  • Disable antivirus temporarily during install
  • Always run setup as Administrator
  • Install .NET Framework 4.5.2 if not present
  • MS SQL Server required for SQL version only

Download & Trial Questions

Common questions before you download — answered.

Is the free trial really free? No credit card needed?
Yes, completely free. The 7-day free trial requires no credit card, no payment details, and no commitment of any kind. Simply download, install and explore all features.
Does the trial have limited features?
No. The trial version is the complete, full-featured software. All modules are unlocked — auto-import, capital gain reports, accounting, ITR reports, family portfolio management and everything else. Nothing is hidden or restricted.
Does i-Record work on Mac or mobile?
Currently, i-Record is available for Windows only (Windows 10 / 11 / Server 2016 / 2022, 64-bit). It does not support Mac, Linux or mobile platforms at this time.
Is my financial data safe? Does it go to the cloud?
Your data is 100% private and stored locally on your own computer. i-Record does not upload any of your financial data to any cloud server. There is zero risk of data breach or unauthorized access.
