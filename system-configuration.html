Installation Guide

Install iRecord in Under 2 Minutes

Getting started is simple. Follow these 4 steps to install and start using iRecord on your Windows PC.

01

Check Requirements

Verify your Windows version and available disk space against the requirements above.

02

Download Installer

Click 'Download Free Trial' and save the iRecord setup file (.exe) to your computer.

03

Run Installer

Double-click the installer and follow the on-screen steps. Installation takes under 2 minutes.

04

Activate & Start

Launch iRecord, enter your activation key or start the free 7-day trial — no internet needed!
